
# Stock & ETF Signal Dashboard

A Python + Streamlit based dashboard that generates BUY/SELL/HOLD signals
for stocks and ETFs using technical indicators.

## Features
- Live stock & ETF data (Yahoo Finance)
- Buy / Sell / Hold signals
- SMA20 & SMA50 crossover strategy
- RSI indicator
- Interactive Streamlit dashboard

## Tech Stack
- Python
- Streamlit
- Pandas, NumPy
- yFinance
- TA indicators

## How to Run
```bash
pip install -r requirements.txt
python -m streamlit run app.py
